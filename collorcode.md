

# 🎨 🎯 Primary Color Palette (Futuristic Red AI Theme)

### 🔴 Primary (Brand Color)

* **#FF3B3B** → Merah utama (energetic, powerful)
* **#E50914** → Merah deep (lebih premium & elegan)

👉 Dipakai untuk:

* CTA button
* Highlight
* Logo accent

---

### 🌑 Dark Base (Background)

* **#0A0A0F** → Hitam kebiruan (futuristik)
* **#111827** → Dark navy (UI modern)
* **#020617** → Super dark (depth / hero section)

👉 Ini penting supaya merah “nyala” 🔥

---

### 🟣 Secondary Accent (Futuristic Feel)

* **#7C3AED** → Ungu neon
* **#22D3EE** → Cyan AI vibe

👉 Dipakai untuk:

* Gradient
* Hover effect
* AI elements

---

### ⚪ Neutral (Text & UI)

* **#F9FAFB** → Putih soft
* **#9CA3AF** → Abu teks sekunder
* **#374151** → Border / divider

---

# 🌈 🔥 Signature Gradient (Wajib untuk Rianpedia)

Gunakan ini biar kelihatan “AI banget”:

### Gradient 1 (Hero / Button)

```
linear-gradient(135deg, #FF3B3B, #7C3AED)
```

### Gradient 2 (Hover Effect)

```
linear-gradient(135deg, #E50914, #22D3EE)
```

### Gradient 3 (Glow Effect)

```
radial-gradient(circle, rgba(255,59,59,0.4), transparent)
```

### 🧩 Card / Panel

### 🌐 Background

* Section alt: #020617

---

### ✨ Highlight AI Element

* Gradient merah → ungu

---

# 🔥 🎯 Gaya Visual yang Akan Terlihat

Dengan kombinasi ini, Rianpedia akan terasa:

* ⚡ Futuristik (karena dark + neon)
* 🔥 Powerful (karena merah dominan)
* 🧠 AI-driven (karena ungu & cyan)
* 💼 Profesional (karena tetap clean & minimal)

---

# 💡 Tips Penting (Biar Nggak “Norak”)

* Jangan terlalu banyak merah → cukup **10–20% saja**
* Gunakan merah hanya untuk:

  * CTA
  * Highlight
  * Active state
* Background tetap gelap → biar elegan
